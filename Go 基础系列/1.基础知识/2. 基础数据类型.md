## 字符串

字符串的两种方式

1. " "

2. `` 反引号   

   > 保存原格式 里面的字符串不会给转义. 比如 \n 也不会

``` go
var str string
// 字符串是双引号
str = "abc"
// 取字符串第一个字符
ch := str[0]
fmt.Println(ch) // 97

// 原样输出
str2 := `hello
	kasd \n`

格式不变,原样输出

```

## 运算符

- Go中的运算符均是从左至右结合

优先级（从高到底）

- `^      !   `                （一元运算符）单个值为一元
- `*       /    %    <<    >>    &      &^`
- `+      -     |      ^ `                               （二元运算符）
- `==   !=   <    <=    >=    >`
- `<- `                         （专门用于channel）
- `&&`
- `||`
