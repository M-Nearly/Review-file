

## virtualenv安装
`pip3 install virtualenv`  

## 创建虚拟环境
``` python
python3 -m venv /path/to/new/virtual/environment
```

## 启动虚拟环境
``` Python
source venv/bin/activate　
```

# virtualenvwrapper基本使用
1. 创建虚拟环境 mkvirtualenv
`mkvirtualenv --python=/usr/local/python3.5.3/bin/python venv`
2. 基本命令
``` python
查看当前的虚拟环境目录
>>> workon
py2 py3

切换到虚拟环境
workon py3

退出虚拟环境
deactivate

删除虚拟环境
rmvirtualenv venv
```
